Furthermore, anyone can go to comments area to leave his comments to the corresponding suggestion.

Create a dynamic front end by using the AngularJS front end framework Run an server-side application with authentication services on Node.js Model and store data in MongoDB Build an RESTful API using Express.js.

Carcassonne

A tile-placement game in which the players draw and place a tile with a piece of southern French landscape on it.

Written 3700 lines of Java code. Using Hashmap to create data structure which implicit a tile in the game. Using state machine class to control the whole process of the game. Using JButton implemented actionlistener to draw tile image on it.

Written all the JUnit test cases.

Save and load game using serializable.
